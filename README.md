# Social Dining App API

## API Documentation

* Endpoints
  * /api/users
    * GET
    * POST
    * PATCH
    * DELETE
    * /:user_id
      * GET
    * /all/events
      * GET
  * /api/events
    * GET
    * POST
    * /:event_id
      * GET
      * PATCH
      * DELETE
  * /api/attendees
    * GET
    * POST
    * DELETE

## Built with
* NodeJS
* ExpressJS
* PostgreSQL
